Tabla Comparativa de Eventos en JavaScript

Evento Descripción Ejemplo de Código
click Se activa cuando el usuario hace clic en un elemento. 
<button onclick="alert('¡Haz hecho clic!')">
    Clic aquí
</button>
mouseover Ocurre cuando el puntero del ratón entra en un elemento. 
<div onmouseover="this.style.backgroundColor='yellow'">
    Pasa el mouse por aquí
</div>
keydown Se activa cuando el usuario presiona una tecla del teclado. 
<input type="text" onkeydown="console.log('Tecla presionada')" />
load Se dispara cuando la página web ha terminado de cargarse. 
<script>
    window.onload = function() {
        alert("Página cargada completamente");
    };
</script>
submit Se produce cuando un formulario es enviado. 
<form onsubmit="alert('Formulario enviado'); return false;">
    <input type="text" required />
    <button type="submit">Enviar</button>
</form>
